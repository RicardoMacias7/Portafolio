Mobile application for Android developed for the Instituto Tecnológico Superior de Guayaquil, aimed at teachers and students. The app allows users to consult digital books like theses and reserve physical books from the library. ...
Mi rol: Diseño y programación del módulo de reservas (CRUD) y desarrollo del apartado de consultas recientes, que muestra el historial de libros consultados. También participé en la implementación de diversas funcionalidades clave para mejorar la experiencia del usuario.

My role: Design and programming of the reservation module (CRUD) and development of the recent consultations section, which shows the history of consulted books. I also participated in the implementation of various key features to improve the user experience.
